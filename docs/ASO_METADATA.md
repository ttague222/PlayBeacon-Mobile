## App Store Category Recommendations

### iOS App Store
- **Primary Category:** Entertainment
- **Secondary Category:** Utilities or Lifestyle

**Rationale:** PlayBeacon is a discovery/recommendation app, not a game itself. The Games category is extremely competitive and Apple may reject if the app doesn't contain playable content.

### Google Play
- **Category:** Entertainment
- **Content Rating:** Everyone

**Rationale:** Content discovery apps fit best in Entertainment. Using Games category risks rejection since users don't play games within the app itself.

## Roblox Trademark Usage Guidelines

### What's PROHIBITED
| Usage | Status |
|-------|--------|
| Using "Roblox" as the **first word** in app name | ❌ Not allowed |
| Using the Roblox **logo** anywhere | ❌ Not allowed |
| Using "Blox" or similar variations in names | ❌ Not allowed |
| Implying official affiliation or endorsement | ❌ Not allowed |

### What's ALLOWED
| Usage | Status |
|-------|--------|
| Descriptive use in app descriptions | ✅ Allowed |
| "for Roblox" phrasing (not as first word) | ✅ Generally allowed |
| iOS Keywords field (hidden, for search only) | ✅ Allowed |
| Stating app helps discover "Roblox games" | ✅ Allowed (descriptive) |

### Required Disclaimer
Include in all store descriptions:
```
DISCLAIMER: PlayBeacon is an unofficial app and is not affiliated with, endorsed by, or sponsored by Roblox Corporation. Roblox and all related titles are trademarks of Roblox Corporation.
```

### Platform-Specific Notes

**Google Play:** More lenient with descriptive trademark use. "Roblox" can appear in short description and full description as long as:
- It's not the first word in the app title
- No Roblox logos are used
- Disclaimer is included
- No implication of official endorsement

**iOS App Store:** Stricter enforcement. Keep "Roblox" out of title and subtitle. Use only in:
- Keywords field (hidden)
- Description body (with disclaimer)
- Promotional text (carefully)
